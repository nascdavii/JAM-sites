# JAM Sites

Landing pages de alta conversão para os serviços da **JAM Refrigeração e Climatização**.

## Páginas

- `index.html` — Menu principal (seleção de serviço)
- `refrigeracao.html` — LP de Refrigeração (geladeira, freezer, máquina de lavar)
- `climatizacao.html` — LP de Climatização (ar-condicionado)

## Setup

```bash
npm install
npm run dev
```

## Stack

- HTML5 + CSS customizado (sem framework UI)
- Vite (dev server + build)
- Design system próprio com CSS variables

## Estrutura

```
├── index.html              # Menu
├── refrigeracao.html       # LP Refrigeração
├── climatizacao.html       # LP Climatização
├── style.css               # Design system compartilhado
└── images/
    ├── hero-refrigeracao.png
    ├── hero-climatizacao.png
    └── tecnico_profissional.png
```
